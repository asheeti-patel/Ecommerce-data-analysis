# 📊 Ecommerce Data Analysis

Data analysis project on a real e-commerce dataset with 9,994 orders,
analyzing sales trends, profit patterns, and customer segments
using Python and Plotly.

---

## 📌 What this project covers

- Monthly Sales Analysis
- Sales by Category and Sub-Category
- Monthly Profit Analysis
- Profit by Category and Sub-Category
- Sales and Profit by Customer Segment
- Sales to Profit Ratio Analysis

---

## 🛠️ Tech Used

- Python 3
- Pandas
- Plotly Express
- Plotly Graph Objects
- Jupyter Notebook

---

## 📈 Key Findings

- November and December had highest sales
- Technology category generated highest profit (50.8%)
- Furniture had lowest profit margin (6.44%)
- Home Office segment had best sales to profit ratio
